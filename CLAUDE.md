# Next.js + Supabase Project

## 프로젝트 개요
Next.js와 Supabase를 활용한 풀스택 앱 프로젝트입니다.

## 기술 스택
- Next.js
- TypeScript
- Supabase
- Tailwind CSS

## 주요 규칙
- 환경변수는 절대 Git에 올리지 않습니다.
- .env.local 파일은 수정 전 주의합니다.
- 기능 개발은 골격 → 공통 → 개별 기능 순서로 진행합니다.
