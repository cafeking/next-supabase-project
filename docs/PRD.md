# PRD: Next.js + Supabase App

## 프로젝트 개요
Next.js와 Supabase를 활용한 풀스택 웹 애플리케이션입니다.

## 목표
인증, 데이터 저장, 기본 UI를 갖춘 확장 가능한 앱 기반을 구축합니다.

## 핵심 기능
- Supabase 인증 연동
- 사용자 로그인/회원가입
- 데이터베이스 CRUD
- 반응형 UI

## 기술 스택
- Next.js
- TypeScript
- Supabase
- Tailwind CSS

## 사용자 스토리
- 사용자는 회원가입하고 로그인할 수 있다.
- 사용자는 자신의 데이터를 생성하고 조회할 수 있다.
- 사용자는 모바일과 데스크톱에서 서비스를 이용할 수 있다.
